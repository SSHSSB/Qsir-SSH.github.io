# 1. 建议
Github因为服务器在国外并且有GFW的屏障，因此在国内访问并不是很顺畅，建议配合加速器使用。推荐一款：Watt Toolkit，原steam++，里面有提供Github加速服务。官网：https://steampp.net/  。
